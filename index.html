LINORIN Changelog

Linked Intelligence Network Orchestrator

v1.6.0 (2026-03-08)
🐛 バグ修正
  • セレクト日本語化: buildFieldHtml のセレクト描画で item.l（日本語ラベル）が無視されていた根本バグを修正。会話モード・通知タイミングなど全フィールドが正しく表示されるように
  • 会話モード pre ブロック: 説明欄の Away / Home / Unified を外出中 / 在宅 / 統合 に日本語化
  • テーマ文字色: calculateAccentTextColor をアクセント輝度基準→背景色輝度基準に変更。ワインなど暗背景+明アクセントで文字が黒くなる問題を解消
  • アクションボタン: .act-btn の背景を透明→テーマカラー薄掛けに変更。ライトテーマで枠だけに見える問題を修正
  • フォントサイズ適用: applyFontSize の全 CSS ルールに !important を追加。inline style による上書きを防止
  • pre フィールド: テキスト色を var(--muted)→var(--text) に変更、font-weight:600 を追加（読みにくい問題を解消）
  • 保存ボタンが「保存中」のまま詰まる: 削除済み要素 f_webui_theme_color の null 参照による TypeError を修正
✨ 新機能 / 改善
  • 時間帯設定: 24 時間を朝/昼/夕方/夜の 4 帯に分けるビジュアルスライダーを追加。境界時刻をドラッグ調整、時間帯ごとにインストラクションを設定可能
  • 時間帯システムプロンプト: getCurrentTimePeriod() を追加。現在時刻に応じた時間帯の指示をシステムプロンプトに自動追記
  • フォントサイズスライダー: A－/A＋ ボタン→スライダーに変更（width:192px）、8 段階（11〜25px）に拡張
  • アップグレードバナー: マトリックスグリーン（#020f02 / #00ff41）固定デザインに変更。どのテーマでも干渉しない
  • テーマプリセット: プルダウン化（webui_selected_theme）。ヘッダーの選択と設定フォームを双方向同期
v1.5.1 (2026-03-04〜06)
✨ 機能追加 / 改善
  • WebUI カラー設定: 背景色・AI 吹き出し・ユーザー吹き出し・ヘッダー・入力バー・アクセント色を設定画面から変更可能に
  • フォント拡張: ヘッダーにフォントファミリー選択（8 種）とサイズ調整を統合
  • 詳細設定トグル: 詳細設定を折りたたんで基本設定だけを表示するトグルを追加
  • ヘルプツールチップ: 各設定項目にヘルプアイコンを追加
  • レスポンシブアイコン: モバイル/タブレット向け自動スケーリングアイコン（5 段階）
  • 必須設定ハイライト: 未入力の必須 API キーを自動ハイライト表示
  • UI 構造の見直し: 頻繁に変更する設定とシステム設定を分離
🐛 バグ修正
  • アップグレードボタン: スコープ外変数参照の ReferenceError で runSetup() が呼ばれない問題を修正
  • ボタン文字色: 白 #fff→ソフトグレー #e0e0ec、text-shadow を追加
  • CSS 整合性: --fg/--bg2 未定義変数を var(--text)/var(--ibg) に修正
v1.5.0 (2026-03-03)
✨ チャネル・モード設定
  • active_mode: away（LINE 等）/ home（WebUI 等）/ unified（全チャネル統合）の 3 モードを追加
  • mode_instruction: Away/Home ごとにシステムプロンプト先頭に挿入する状況説明を設定可能
  • channel_instruction: LINE / Slack / Telegram / WebUI ごとに補足指示を設定可能
  • Unified モード: 全チャネルの会話履歴を統合参照（72h / 最大 50 件）
✨ RAG UI 改善
  • キーワード直接編集: RAG エントリをキー・コンテンツの 2 フィールド構成に刷新
  • 追加・削除ボタン: UI からエントリを追加・削除可能、セクション部分再描画に対応
✨ その他
  • WebUI event_queue 統合: WebUI メッセージを event_queue に記録し監査ログを統一
  • type:'pre' フィールド: 等幅フォントの ASCII アート表示フィールドを追加

v1.4.4〜10 (2026-03-01)
🐛 アップグレード・設定 UI 安定化

  • Gemini 無料モデルフィルタ: GEMINI_FREE_TIER_ONLY 設定を追加。Flash / Gemma / Lite 系を無料判定
  • 設定 UI コマンドボタン整理: 今すぐ話しかける・日記生成・モデル更新に絞り込み。その他はデバッグセクションへ
