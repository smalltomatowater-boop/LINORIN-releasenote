v1.10.10 (2026-03-26)

v1.10.9 (2026-03-26)
🔧 バグ修正まとめ
  • fixWebUI + unified モードで「今すぐ生成」が機能しなかった問題を修正: getAllUserStates が allowed_line_users フィルタで WebUI ユーザーを除外、generateDailyDiaryForAllUsers が LINE platform で通知していた 2 つのバグを修正。
  • fixWebUI + unified モードでのプッシュが LINE で上書きされる問題を修正: scheduledCheck / generateLongTermMemoryForAllUsers に effectivePlatform パターンを適用。
  • fix会話ゼロの日も日記を生成するよう変更: 会話ログが 0 件の場合、「今日は会話がなかった」待ち日記プロンプトに自動切替して生成。
  • fixカスタム壁紙が再読み込み後に消える問題を修正: アップロード画像を Drive に永続保存し、localStorage をキャッシュとして利用するハイブリッド方式に変更。
  • fixさくらのAIクラウドのモデルが WebUI に表示されなかった問題を修正: SAKURA_MODEL / SUB_SAKURA_MODEL の選択肢をスキーマに直接埋め込み、初回から選択可能に。
  • fixDrive API 経由の PDF テキスト抽出が機能していなかった問題を修正: isDriveApiAvailable() の Drive v2 フィールド名誤り（files→items）を修正。
  • fixテラリウムがユーザー発言に反応しない問題を修正: Google Sheets が "TRUE" を boolean true に自動変換するため is_bot 判定が全 bot メッセージで false になり、hourly limit を使い切ってからユーザーに無反応になっていた。
v1.10.8 (2026-03-26)
🔧 バグ修正 & 内部整理
  • fixさくらのAIクラウドのモデルが WebUI に表示されなかった問題を修正: SAKURA_MODEL / SUB_SAKURA_MODEL の選択肢をスキーマに直接埋め込み、初回から選択可能に。
  • fixDrive API 経由の PDF テキスト抽出が機能していなかった問題を修正: isDriveApiAvailable() の Drive v2 フィールド名誤り（files→items）を修正。extractTextViaDrive() を Drive.Files.insert() 直接変換方式に簡略化。
  • fixテラリウムがユーザー発言に反応しない問題を修正: Google Sheets が "TRUE" を boolean true に自動変換するため is_bot 判定が全 bot メッセージで false になり、bot 同士の連鎖で hourly limit を消費してからユーザーに無反応になっていた。
  • fixローカルLLMサーバー停止中のタイムアウト問題を修正: LOCAL_LLM_ENABLED=FALSE でサーバー接続をスキップしてフォールバックへ即切替。
  • refactorLINORIN.js のセクション順を整理: SETUP を先頭近く・buildSystemText を MESSAGE BUILD 直前・エントリーポイントをまとめ・デバッグ関数を debug_tools.js へ集約。
  • fixWebUI + unified モードで「今すぐ生成」が機能しなかった問題を修正: getAllUserStates が allowed_line_users フィルタで WebUI ユーザーを除外、generateDailyDiaryForAllUsers が LINE platform で通知していた 2 つのバグを修正。
  • fixWebUI + unified モードでのプッシュが LINE で上書きされる問題を修正: scheduledCheck / generateLongTermMemoryForAllUsers に effectivePlatform パターンを適用。
  • fix日記の二重生成を防止: ScriptProperties による当日分のデデュープロックを追加（WebUI 再生成機能は対象外）。
